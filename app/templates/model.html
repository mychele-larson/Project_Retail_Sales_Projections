Model

In this project, we use the Facebook Prophet model for time series forecasting. Facebook Prophet is a powerful and flexible machine learning library designed to handle a wide range of time series data. It is particularly effective at modeling seasonal trends and handling missing data.

Some of the key features of Facebook Prophet include:

  • Automatic detection of seasonality and trend components
  • Handling of missing data and outliers
  • Flexible model customization
  • Easy integration with Python and R
